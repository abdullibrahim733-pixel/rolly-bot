# CAD Design

The CAD files for Rolly Bot are hosted on Onshape.

## View Design

Use the link below to view and edit the CAD model in Onshape:

**Onshape URL:** https://cad.onshape.com/documents/393bdf646ad824c0b8550b00/w/9274fd8af6c662434374118b/e/fe61023ea63f560d9cedba20?renderMode=0&uiState=69e41e73e44967b82eb4f5f9

## Building the Design

1. Open the Onshape link above
2. Sign in to your Onshape account
3. Use the standard parts and assemblies to export STEP files if needed
4. Generate manufacturing files (PDF, DXF) for fabrication

## Notes

- Onshape is a cloud-based CAD platform that runs in the browser
- You may need an Onshape account to access and edit the design
- Export options available: STEP, IGES, STL, and various manufacturing formats
